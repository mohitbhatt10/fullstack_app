Admin Portal Overview

The Admin portal enables school administrators to manage core academic and operational data.

Main Capabilities

  • Manage students, teachers, courses, schedules and exam types.
  • Approve profile change requests.
  • Monitor attendance & marks summaries (future dashboards).
  • Configure academic sessions and system settings.

Typical Daily Actions

  • Create / update course schedules.
  • Enroll new students & assign courses to teachers.
  • Review profile update requests.
  • Export reports.

This page is publicly viewable so prospective users understand admin responsibilities.
