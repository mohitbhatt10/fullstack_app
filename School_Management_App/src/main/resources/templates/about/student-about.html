Student Portal Overview

The Student portal provides learners with transparent access to their academic journey.

Main Capabilities

  • View enrolled courses and session details.
  • Track attendance records and percentages.
  • View marks, pass rates and exam performance.
  • Access profile and request updates (if enabled).

Engagement Benefits

  • Immediate insight into attendance trends.
  • Clear breakdown of marks by exam type.
  • Consistent UI shared across user roles.

This page is public to show prospective students how the system supports them.
